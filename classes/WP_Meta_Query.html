List of metadata queries.

$queries : array

A single query is an associative array: - 'key' string The meta key - 'value' string|array The meta value - 'compare' (optional) string How to compare the key to the value. Possible values: '=', '!=', '>', '>=', '<', '<=', 'LIKE', 'NOT LIKE', 'IN', 'NOT IN', 'BETWEEN', 'NOT BETWEEN'. Default: '=' - 'type' string (optional) The type of the value. Possible values: 'NUMERIC', 'BINARY', 'CHAR', 'DATE', 'DATETIME', 'DECIMAL', 'SIGNED', 'TIME', 'UNSIGNED'. Default: 'CHAR'
