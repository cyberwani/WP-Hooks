List of taxonomy queries.

$queries : array

A single taxonomy query is an associative array: - 'taxonomy' string The taxonomy being queried - 'terms' string|array The list of terms - 'field' string (optional) Which term field is being used. Possible values: 'term_id', 'slug' or 'name' Default: 'term_id' - 'operator' string (optional) Possible values: 'AND', 'IN' or 'NOT IN'. Default: 'IN' - 'include_children' bool (optional) Whether to include child terms. Default: true

since 3.1.0
